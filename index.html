What Is a CSV File?

CSV = Comma-Separated Values

  • Plain text format for tabular data
  • Each line is a row
  • Commas separate values

Typical Structure

  • Row 1: header names
  • Rows 2+: actual records
  • Easy to share across tools

Think of CSV as a spreadsheet saved as plain text.

What Does CSV Look Like?

name,age,grade
Alice,18,90
Bob,19,85
Carol,18,92
Diana,20,78
Line Meaning
Line 1 Header row (column names)
Lines 2+ Data rows with matching column order

Which block is properly formatted CSV?

B)
name,age,grade
Alice,18,90

Reveal: Why B Is Correct

B follows standard CSV rules: comma-separated values with a header row.

Why A is wrong

It uses pipes |, not commas.

Why C is wrong

It uses key-value text, not CSV columns.

csv.reader and Header Skipping

import csv

with open('students.csv', 'r', newline='') as file:
    reader = csv.reader(file)
    next(reader)  # skip header row
    for row in reader:
        name = row[0]
        age = row[1]
        grade = row[2]
        print(f"{name} is {age} years old with grade {grade}.")
Each row from csv.reader is a list of strings.

DictReader vs reader

import csv

with open('students.csv', 'r', newline='') as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(row['name'], "-", row['grade'])
Feature csv.reader csv.DictReader
Row type list dict
Access style row[0] row['name']
Header handling manual next(reader) automatic

Writing CSV: writer and writerows

import csv

students = [
    ['Alice', 18, 90],
    ['Bob', 19, 85],
    ['Carol', 18, 92],
]

with open('output.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['name', 'age', 'grade'])
    writer.writerows(students)
Use writerow for one row, writerows for many rows.

DictWriter with fieldnames

import csv

students = [
    {'name': 'Alice', 'age': 18, 'grade': 90},
    {'name': 'Bob', 'age': 19, 'grade': 85},
]

fields = ['name', 'age', 'grade']

with open('output.csv', 'w', newline='') as file:
    writer = csv.DictWriter(file, fieldnames=fields)
    writer.writeheader()
    writer.writerows(students)

Common Errors and Fixes

Error Cause Fix
FileNotFoundError Wrong file path Check name and location
Extra blank lines Missing newline='' Add newline='' in open()
Numbers read as strings csv returns text values Convert with int() or float()
DictWriter column mismatch fieldnames mismatch Align dict keys and fieldnames

Real-World Example: Grade Report

import csv

totals, count = 0, 0
results = []

with open('grades.csv', 'r', newline='') as f:
    reader = csv.DictReader(f)
    for row in reader:
        score = int(row['score'])
        totals += score
        count += 1
        results.append({
            'name': row['name'],
            'score': score,
            'passed': 'Yes' if score >= 75 else 'No'
        })

with open('report.csv', 'w', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=['name', 'score', 'passed'])
    writer.writeheader()
    writer.writerows(results)

print(f"Class average: {totals / count:.2f}")

Quick Reference Cheat Sheet

import csv

# read: list rows
with open('file.csv', 'r', newline='') as f:
    for row in csv.reader(f):
        print(row)

# read: dict rows
with open('file.csv', 'r', newline='') as f:
    for row in csv.DictReader(f):
        print(row['column'])

# write: list rows
with open('file.csv', 'w', newline='') as f:
    w = csv.writer(f)
    w.writerow(['col1', 'col2'])
    w.writerows(data)

with open('file.csv', 'w', newline='') as f:
    w = csv.DictWriter(f, fieldnames=['col1', 'col2'])
    w.writeheader()
    w.writerows(data)
